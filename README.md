# Radian Gridsome Site

A Personal Portfolio site using Gridsome as a static site generator.

### 1. Install Dependencies

`npm install`

### 2. Fire up a local dev server

`gridsome develop`

### 3. Build static files

`gridsome build`

## Pages
 - Home
 - About
   - Background
   - Current Stack
   - People I follow
 - Portfolio
 - Feed
 - Contact
